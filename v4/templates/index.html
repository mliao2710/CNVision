Examples ▸

Example 1: Exon Number Mode
Input Mode: Exon Numbers
Gene Symbol: SPOUT1
First Exon: 8
Last Exon: 11
CNV Type: Duplication
ℹ️ Use this mode when you know which exons are affected (e.g., from clinical reports stating "exons 8-11")
Example 2: Transcript/cDNA Notation
Input Mode: Transcript/cDNA Notation
cDNA Notation: FBN1:c.5065_5546del
Alternative format: NM_001406716.1:c.5065_5546del
CNV Type: Deletion (auto-detected from 'del')
ℹ️ Use this format when you have HGVS cDNA notation. Accepts both:
• Gene symbol format: FBN1:c.5065_5546del
• Transcript ID format: NM_001406716.1:c.5065_5546del
The 'del' or 'dup' suffix will auto-detect the CNV type.
Example 3: Genomic Coordinates (Simple Format)
Input Mode: Genomic Coordinates
Genomic Input: DEL chr17:43,086,436-43,095,443
Genomic Build: GRCh38
Gene (hint): BRCA1
CNV Type: Deletion
ℹ️ Use this format when you have coordinates from sequencing data. Can use formats like:
• chr17:43,086,436-43,095,443
• DEL chr17:43,086,436-43,095,443
• 17:43086436-43095443 (without commas)
Example 4: Genomic Coordinates (ISCN/Microarray Format)
Input Mode: Genomic Coordinates
Genomic Input: arr[GRCh37]15q21.1(48,741,090_48,756,096)x1
Genomic Build: GRCh37 (auto-detected from input)
Gene (hint): FBN1
CNV Type: Deletion (auto-detected: x1 = deletion)
ℹ️ Paste ISCN notation directly from microarray reports. The tool will auto-detect:
• Genomic build (GRCh37 vs GRCh38)
• CNV type (x1 = deletion, x3 = duplication)
• Chromosome and coordinates
Format: arr[Build]CHROMOSOMEband(start_end)xCOPY
⚠️ Important Notes
Genomic Build: Make sure to select the correct build (GRCh37/hg19 vs GRCh38/hg38). Using the wrong build will give incorrect results!
Gene Hint: For genomic coordinates, providing a gene symbol helps ensure accurate results, especially for regions with multiple overlapping genes.
Supported Genes: This tool uses MANE Select transcripts & NCBI refseq isoforms. Most clinically relevant genes are included.
